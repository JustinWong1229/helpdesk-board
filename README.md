# Help Desk Board

Setup steps:
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


Features Checklist:
Project is organized into components
Multiple componets
Board holds the main state for all tickets
UseEffecr used to fetch a API
Tickets render with title, status, priority and update time
